== Installation ==

### Automatic installation

1. Search for *Surbma | Secure Login* through Plugins > Add New interface.
2. Find the plugin box of *Surbma | Secure Login* and click on the "Install Now" button.
3. Then activate the *Surbma | Secure Login* plugin.
4. That's it. :)

### Manual installation

1. Download [Surbma | Secure Login](https://downloads.wordpress.org/plugin/surbma-secure-login.zip)
2. Upload *Surbma | Secure Login* through Plugins > Add New > Upload interface or upload surbma-secure-login folder to the "/wp-content/plugins/" directory.
3. Activate the *Surbma | Secure Login* plugin through the "Plugins" menu in WordPress.
4. That's it. :)
